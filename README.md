
## Rick and morty DB
App demo can be viewed here : https://ricknmorty-parimal.netlify.app/
Note: Development was done on windows, there could be issues running on ubuntu

Features:
- Uses ESLint for consistent coding style.
- Promise.all utilized to avoid network waterfall.
- Responsive design using Tailwind css
- Pagination
